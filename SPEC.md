# buzhidao edu - 规格文档

## 1. 项目概述

- **项目名称**: buzhidao edu
- **项目类型**: 单页Web应用 (HTML5)
- **核心功能**: 多功能理科学习平台，支持数学函数图像、几何画板、化学式输入
- **目标用户**: 学生、教师、理科爱好者
- **技术特点**: 离线运行，无需服务器，纯前端实现

## 2. UI/UX 规格

### 布局结构

- **页面结构**: 单一页面，垂直布局
  - 顶部：网站标题与简洁描述
  - 中部：画布区域（函数图像显示）
  - 底部：输入控制区（函数输入框、数学符号按钮、工具按钮）
- **响应式断点**:
  - 移动端: < 640px（垂直堆叠，控件紧凑）
  - 平板: 640px - 1024px
  - 桌面: > 1024px

### 视觉设计

- **主色调**:
  - 主色: #0D9488 (teal-600)
  - 浅色: #14B8A6 (teal-500)
  - 深色: #0F766E (teal-700)
  - 背景: #F8FAFC (slate-50)
  - 卡片背景: #FFFFFF
  - 文字主色: #1E293B (slate-800)
  - 文字次色: #64748B (slate-500)
- **字体**:
  - 标题: "Outfit", sans-serif (Google Fonts)
  - 正文: "IBM Plex Sans", sans-serif
  - 等宽(数学): "JetBrains Mono", monospace
- **间距系统**: 4px 基础单位 (0.25rem)
- **视觉效果**:
  - 卡片阴影: 0 4px 6px -1px rgba(0,0,0,0.1)
  - 按钮悬停: scale(1.02) + 阴影增强
  - 过渡动画: 0.2s ease-out

### 组件列表

1. **标题组件**: 网站名称 + 副标题
2. **画布组件**:
   - HTML5 Canvas
   - 显示坐标轴、网格、函数曲线
   - 支持拖拽查看坐标
3. **函数输入框**:
   - 文本输入框
   - 实时预览
   - 支持表达式如: sin(x), x^2, abs(x), sqrt(x) 等
4. **数学符号键盘**:
   - 常用符号按钮: π, e, √, ^, sin, cos, tan, log, ln, (, ), +, -, *, /, x, =
   - 点击后插入到输入框
5. **控制按钮**:
   - 绘制按钮（主题色）
   - 清除按钮
   - 下载图像按钮
6. **预设函数选择器**: 下拉菜单可选常用函数

## 3. 功能规格

### 核心功能

#### 数学模块
1. **函数图像生成**
   - 支持基本运算: +, -, *, /, ^
   - 支持数学函数: sin, cos, tan, asin, acos, atan, abs, sqrt, log, ln, exp
   - 支持数学常量: pi, e
   - 默认显示范围: x: [-10, 10], y: [-10, 10]

2. **多函数支持**
   - 可同时添加多个函数
   - 每个函数独立颜色设置
   - 每个函数独立线宽设置
   - 每个函数独立定义域设置
   - 显示函数列表

3. **函数分析**
   - 计算最小值/最大值
   - 计算Y轴截距
   - 计算零点
   - 显示函数类型
   - 显示导数近似

4. **交点计算**
   - 自动计算函数交点
   - 交点可视化标记

5. **内置数学输入法**
   - 点击符号按钮输入
   - 预设函数下拉选择
   - 快捷键支持

#### 几何模块
1. **几何工具**
   - 点工具
   - 直线工具
   - 线段工具
   - 射线工具
   - 圆工具

#### 化学模块
1. **化学式输入**
   - 元素周期表快捷输入
   - 常用化学式预设
   - 下标/上标自动转换

#### 系统功能
1. **主题切换**
   - 深色模式
   - 浅色模式
   - 主题记忆

2. **响应式设计**
   - 桌面端: 双栏布局
   - 平板/移动端: 单栏布局

3. **图像下载**
   - 保存为PNG格式
   - 自动时间戳命名

4. **坐标系统**
   - XY轴范围自定义
   - 网格开关
   - 坐标轴开关
   - 刻度标签开关
